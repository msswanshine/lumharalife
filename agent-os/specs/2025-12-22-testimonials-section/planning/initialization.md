# Initial Description: Testimonials Section

## Feature Description
Create a testimonials section on the homepage that showcases client feedback and social proof, building trust with visitors. The section should display testimonials from clients who have worked with Amy Swan, including their names, titles/roles, and testimonial text.

## Key Requirements
- Display testimonials in a visually appealing layout
- Include client names and titles/roles
- Support for testimonial text content
- Support for video testimonials (if available)
- Match tranquil, minimal aesthetic
- Positioned on homepage (after coming soon section, before contact)

## Content Source
Testimonials to be copied from https://flowinallthings.com/ including any video content.
