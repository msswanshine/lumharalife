Accessibility Statement

Last Updated: December 2025

At Lumhara, I am committed to ensuring digital accessibility for people with disabilities. I am continually improving the user experience for everyone and applying the relevant accessibility standards to achieve these goals.

Conformance Status

I strive to conform to the Web Content Accessibility Guidelines (WCAG) 2.2 at Level AA. These guidelines explain how to make web content more accessible for people with disabilities and user-friendly for everyone.

While I work to achieve and maintain WCAG 2.2 Level AA compliance, I recognize that accessibility is an ongoing effort. I welcome your feedback on the accessibility of this website.

Accessibility Features

This website includes the following accessibility features:

  • Semantic HTML: I use proper HTML5 semantic elements (header, nav, main, section, article, footer) to provide clear structure and meaning to assistive technologies.
  • Keyboard Navigation: All interactive elements can be accessed and operated using only a keyboard. Focus indicators are clearly visible throughout the site.
  • Screen Reader Support: I use ARIA labels and attributes to provide additional context for screen reader users, including descriptive labels for buttons, links, and form inputs.
  • Alt Text for Images: All meaningful images include descriptive alternative text. Decorative images are marked appropriately to be ignored by screen readers.
  • Color Contrast: Text and interactive elements meet WCAG 2.2 Level AA contrast requirements (minimum 4.5:1 for normal text, 3:1 for large text).
  • Responsive Design: The website is fully responsive and works across desktop, tablet, and mobile devices.
  • Text Resizing: Text can be resized up to 200% without loss of functionality using browser zoom controls.
  • Reduced Motion: Animations respect user preferences for reduced motion. Users who prefer reduced motion will see simplified or static versions of animated content.
  • Form Labels: All form inputs have associated labels, making forms accessible to screen reader users and providing clear instructions.
  • Focus Management: Focus is properly managed in interactive components like modals, ensuring keyboard users can navigate effectively.
  • Descriptive Link Text: Links are written to be descriptive and make sense out of context, avoiding generic text like "click here."
  • Page Titles: Each page has a unique, descriptive title that helps users understand the page content.

Known Limitations

Despite my best efforts to ensure accessibility, there may be some limitations:

  • Some third-party content (such as embedded YouTube videos) may have varying levels of accessibility. While I strive to ensure all embedded content meets accessibility standards, I cannot guarantee the accessibility of third-party content.
  • If you encounter accessibility issues with third-party content, please let me know, and I will work to find accessible alternatives when possible.

Assistive Technologies

This website has been tested with the following assistive technologies:

  • Screen readers (NVDA, JAWS, VoiceOver)
  • Keyboard-only navigation
  • Browser zoom up to 200%
  • High contrast mode

Feedback and Contact

I welcome your feedback on the accessibility of the Lumhara website. Accessibility is an ongoing commitment, and I regularly review and update this website to improve accessibility and ensure compliance with current standards.

If you encounter accessibility barriers or have suggestions for improvement, please contact me:

  • Email: Contact Form
