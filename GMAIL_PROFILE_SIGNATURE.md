# Gmail Profile Picture & Signature Setup

## Part 1: Add Profile Picture

### Step 1: Add Profile Picture to Google Account

1. **Go to:** myaccount.google.com
2. **Click on your profile picture** (top right)
3. **Click "Manage your Google Account"**
4. **Click on your profile picture again** (or go to Personal info → Photo)
5. **Click "Add profile picture"** or the camera icon
6. **Upload your photo** (headshot or professional photo)
7. **Crop and adjust as needed**
8. **Click "Set as profile photo"**

**Note:** This profile picture will show for your entire Google account, but it's the best way to have a professional image associated with your emails.

## Part 2: Create Professional Signature for Lumhara

### Step 1: Create Signature in Gmail

1. **Open Gmail**
2. **Settings** (gear icon → See all settings)
3. **General tab**
4. **Scroll down to "Signature" section**

### Step 2: Create New Signature

1. **Click "Create new"** (if you don't have one)
2. **Name it:** `Lumhara Business` or `Lumhara`
3. **In the signature editor, add:**

```
Amy Swan
Human Systems Guide & Coach
Lumhara

hello@lumharalife.com
lumharalife.com
```

### Step 3: Add Profile Image to Signature (Optional)

You can add your headshot image to the signature:

1. **In the signature editor, place your cursor where you want the image**
2. **Click the image icon** (or Insert → Image)
3. **Upload your headshot** (or use a URL if hosted online)
4. **Resize as needed** (recommended: 100-150px height)
5. **Add alt text:** "Amy Swan, Human Systems Guide & Coach"

**Example signature with image:**
```
[Amy's Headshot Image - 120px height]

Amy Swan
Human Systems Guide & Coach
Lumhara

hello@lumharalife.com
lumharalife.com
```

### Step 4: Link Image (If Using URL)

If you want to use your website headshot:
1. **Host the image on your website:** `assets/images/about-me_headshot.png`
2. **Use the full URL:** `https://lumharalife.com/assets/images/about-me_headshot.png`
3. **Insert image in signature using the URL**

### Step 5: Configure When to Use Signature

1. **Under "Signature defaults":**
   - **For new emails:** Select your "Lumhara" signature
   - **For replies/forwards:** Select your "Lumhara" signature

2. **Important:** Gmail doesn't automatically use different signatures for different "Send mail as" addresses, but you can:
   - **Manually select signature** when composing
   - **Or set Lumhara as default** and manually change for personal emails

## Part 3: Advanced - Different Signatures for Different Addresses

Gmail doesn't automatically switch signatures based on "Send mail as" address, but you can:

### Option A: Manual Selection (Easiest)

1. **When composing an email**
2. **Click the signature dropdown** (bottom of compose window)
3. **Select "Lumhara" signature** when sending from `hello@lumharalife.com`
4. **Select personal signature** (or none) for personal emails

### Option B: Use Gmail Add-on (Advanced)

- **WiseStamp** (free tier available) - Can auto-switch signatures
- **Signature Creator** - Browser extension
- **Or just remember to select the right signature manually**

## Troubleshooting

### Signature not appearing?
- Make sure you selected it in the compose window
- Check signature is set as default in settings
- Try refreshing Gmail

### Image not showing?
- Make sure image is publicly accessible (if using URL)
- Check image size (keep under 200KB)
- Try uploading directly to Gmail instead of using URL

### Want different signatures for different addresses?
- Gmail doesn't auto-switch, but you can manually select
- Or use a browser extension/add-on
