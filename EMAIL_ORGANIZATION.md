# Email Organization Guide: Separating Lumhara Emails

This guide helps you organize and differentiate Lumhara business emails from your personal emails.

## Option 1: Gmail Filters & Labels (Recommended for Gmail)

### Step 1: Create a Label for Lumhara Emails

1. **Open Gmail**
2. **Go to Settings** (gear icon → See all settings)
3. **Click "Labels" tab**
4. **Click "Create new label"**
5. **Name it:** `Lumhara` or `Business`
6. **Click "Create"**

### Step 2: Create a Filter

1. **In Gmail Settings, go to "Filters and Blocked Addresses" tab**
2. **Click "Create a new filter"**
3. **In "To" field, enter your Lumhara email:** `hello@lumharalife.com` (or whatever address you created)
4. **Click "Create filter"**
5. **Check these options:**
   - ✅ Apply the label: `Lumhara` (select the label you created)
   - ✅ Skip the Inbox (Archive it) - *Optional: Only if you want Lumhara emails in a separate folder*
   - ✅ Mark as important - *Optional*
   - ✅ Always mark it as read - *Optional: Only if you want to review later*
6. **Click "Create filter"**

### Step 3: Set Up a Separate Folder View

1. **In Gmail left sidebar, click your "Lumhara" label**
2. **All Lumhara emails will appear there**
3. **You can also create a "Lumhara" folder in your email client**

### Step 4: Create a Professional Signature

1. **Gmail Settings → General tab**
2. **Scroll to "Signature"**
3. **Create a new signature:**
   ```
   Amy Swan
   Human Systems Guide & Coach
   Lumhara
   
   hello@lumharalife.com
   lumharalife.com
   ```
4. **Set it to use for new emails**
5. **Optionally, create a second signature for personal emails**

## Option 2: Apple Mail (macOS/iOS)

### Step 1: Create a Smart Mailbox

1. **Open Mail app**
2. **Mailbox → New Smart Mailbox**
3. **Name it:** `Lumhara`
4. **Set condition:** `To` contains `hello@lumharalife.com`
5. **Click "OK"**

### Step 2: Create Rules

1. **Mail → Preferences → Rules**
2. **Click "+" to add a rule**
3. **Name:** `Lumhara Emails`
4. **Conditions:**
   - `To` contains `hello@lumharalife.com`
5. **Actions:**
   - Move message to mailbox: `Lumhara` (create if needed)
   - Or: Set background color (choose a color like green)
6. **Click "OK"**

## Option 3: Outlook

### Step 1: Create a Folder

1. **Right-click your inbox**
2. **New Folder**
3. **Name it:** `Lumhara`

### Step 2: Create a Rule

1. **Home tab → Rules → Manage Rules & Alerts**
2. **New Rule**
3. **Start from a blank rule → Check messages when they arrive**
4. **Conditions:**
   - `To` contains `hello@lumharalife.com`
5. **Actions:**
   - Move to folder: `Lumhara`
   - Or: Categorize with color
6. **Finish**

## Option 4: Use a Separate Email Client/App

### Option A: Use Gmail's "Send mail as" Feature

1. **Gmail Settings → Accounts and Import**
2. **"Send mail as" → Add another email address**
3. **Enter:** `hello@lumharalife.com`
4. **When replying, choose which address to use**

### Option B: Use a Separate App

- **Spark** (iOS/Mac): Can separate accounts
- **Airmail**: Multiple account support
- **Apple Mail**: Multiple accounts with separate inboxes

## Best Practices

### 1. **Visual Differentiation**
- Use different colors/labels for business vs personal
- Set up different notification sounds (if desired)
- Use different folders/labels

### 2. **Professional Signature**
- Always use your Lumhara signature when replying to business emails
- Include your website and contact info

### 3. **Separate Response Times**
- Set expectations: "I check business emails during business hours"
- Use auto-responders if needed

### 4. **Email Organization**
- Archive personal emails regularly
- Keep Lumhara emails organized by client/project
- Use stars/flags for important business emails
